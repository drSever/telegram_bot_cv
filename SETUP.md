# Быстрая настройка бота

## 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

## 2. Настройка Telegram бота
1. Найдите @BotFather в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен

## 3. Настройка токена
```bash
# Отредактируйте файл .env
nano .env

# Замените your_bot_token_here на ваш реальный токен
BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

## 4. Тестирование
```bash
# Тест детектора
python test_detection.py

# Тест с изображением
python test_real_image.py
```

## 5. Запуск бота
```bash
# Рекомендуемый способ (с проверками)
python run_bot.py

# Альтернативный способ
python bot.py
```

## Что происходит при первом запуске:
1. Автоматически скачивается модель YOLOv8n (~6MB)
2. Инициализируется детектор объектов
3. Бот становится доступен в Telegram

## Использование:
1. Найдите вашего бота в Telegram
2. Отправьте `/start`
3. Загрузите фотографию
4. Получите результаты детекции
5. Выберите классы для сегментации

## Поддерживаемые команды:
- `/start` - начать работу
- `/help` - справка
- Отправка фото - анализ изображения
- `car person bicycle` - выбор классов
